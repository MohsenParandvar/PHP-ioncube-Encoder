# PHP-ioncube-Encoder
## a simple tool for ionCube encrypt PHP code writen in python

**Notice : this repository working with [piman.ir](https://piman.ir)**
### Dependencies:
- Python3
- zipfile (python lib)
- ntpath (python lib)
- validators (python lib)

Usage:
```
python3 encoder.py File.php
```
